At A Glance

What this project demonstrates

The repository is more than a notebook. It packages the full ML lifecycle: investigation, preprocessing, model selection, artifact persistence, and a shareable frontend for prediction.

EDA

Checked missing values, duplicates, category distributions, outliers, and feature relationships in notebook/1 . EDA STUDENT PERFORMANCE .ipynb.

Training

Benchmarked multiple regression algorithms with grid search and saved the best model plus the preprocessor for consistent inference.

Deployment

Served the predictor through Flask templates and kept the app easy to run locally with a simple Python entrypoint.
